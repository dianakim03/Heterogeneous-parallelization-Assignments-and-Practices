# Лабораторная работа №1  
## Тема: Введение в основы C/C++ и структуры данных  

## Цель работы
Целью данной практической работы является изучение работы с динамической
памятью и указателями в языке C++, а также ознакомление с основами
параллельного программирования с использованием технологии OpenMP.
В рамках работы рассматривается вычисление среднего значения элементов
динамического массива последовательным и параллельным способами.

## Выполненная часть работы

«Динамическая память и указатели»**.

## Описание выполненного задания (Часть 3)
В рамках части 3 была реализована программа, которая:
- создаёт динамический массив с использованием указателей;
- заполняет массив случайными целыми числами;
- вычисляет среднее значение элементов массива последовательным способом;
- вычисляет среднее значение элементов массива параллельно с использованием OpenMP;
- корректно освобождает выделенную динамическую память.

## Этапы работы программы
1. Создание динамического массива с помощью указателей  
2. Заполнение массива случайными числами  
3. Последовательное вычисление среднего значения элементов массива  
4. Параллельное вычисление среднего значения элементов массива с использованием OpenMP  
5. Освобождение динамически выделенной памяти  

## Используемые технологии
Visual Studio 2022
Язык программирования C++
Технология параллельного программирования OpenMP 
Библиотека `<chrono>` для измерения времени выполнения
