# Практическая работа №2  
## Тема: Параллельная реализация простых алгоритмов сортировки с использованием OpenMP

## Цель работы
Целью данной практической работы является освоение параллельного
программирования на CPU с использованием технологии OpenMP, а также
изучение и реализация базовых алгоритмов сортировки.
В ходе работы проводится сравнение производительности последовательных
и параллельных реализаций алгоритмов сортировки.

## Описание работы
В рамках практической работы были реализованы простые алгоритмы сортировки:
сортировка пузырьком, сортировка выбором и сортировка вставкой.
Каждый алгоритм реализован в двух вариантах:
последовательном и параллельном с использованием директив OpenMP.
Для анализа эффективности параллелизации измеряется время выполнения
каждого алгоритма сортировки при различных размерах массива.

## Реализованные алгоритмы сортировки
В работе реализованы следующие алгоритмы:
- Сортировка пузырьком (Bubble Sort)
- Сортировка выбором (Selection Sort)
- Сортировка вставкой (Insertion Sort)
Для каждого алгоритма выполнена последовательная и параллельная реализация.

## Параллельная реализация
Для распараллеливания вычислений используется технология OpenMP.
В программе применяются следующие директивы:

- `#pragma omp parallel for` — для распараллеливания внешних циклов
- механизмы синхронизации для корректной работы с общими данными

Отмечается, что не все алгоритмы сортировки одинаково хорошо подходят
для параллельной реализации из-за зависимости между итерациями.

## Этапы работы программы
1. Создание массива заданного размера  
2. Заполнение массива случайными значениями  
3. Последовательная сортировка массива выбранным алгоритмом  
4. Параллельная сортировка массива с использованием OpenMP  
5. Измерение времени выполнения последовательной и параллельной реализаций  
6. Сравнение полученных результатов  

## Используемые технологии
Visual Studio 2022
Язык программирования C++  
Технология параллельного программирования OpenMP  
Библиотека `<chrono>` для измерения времени выполнения

## Вывод
В ходе выполнения практической работы были изучены особенности
последовательной и параллельной реализации простых алгоритмов сортировки.
Проведённое сравнение показало, что параллелизация может повысить
производительность, однако эффективность зависит от выбранного алгоритма
и размера обрабатываемого массива.
