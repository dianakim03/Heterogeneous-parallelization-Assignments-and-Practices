# Practice 9  — Распределенная обработка данных с использованием MPI

## Цель
Реализовать 3 распределённые задачи с использованием MPI и проверить работу программы при разном количестве процессов `-np`.

Задачи:
1) Распределённое вычисление среднего и стандартного отклонения (MPI_Scatterv + MPI_Reduce)  
2) Распределённое решение СЛАУ методом Гаусса (MPI_Scatter + MPI_Bcast + MPI_Gather)  
3) Параллельный поиск кратчайших путей (Флойд–Уоршелл) (MPI_Scatter + MPI_Allgather + MPI_Gather)

## Как работает программа

### Общая логика
- `rank 0` готовит входные данные (массив / матрицу / граф)
- данные распределяются между процессами
- каждый процесс делает локальные вычисления
- результаты собираются обратно
- время измеряется через `MPI_Wtime()`

---

## Task 1 — Mean + StdDev (MPI_Scatterv + MPI_Reduce)
1) `rank 0` создаёт массив `N` случайных `double`  
2) массив делится между процессами через `MPI_Scatterv` (учитывается остаток `N % size`)  
3) каждый процесс считает:
   - сумму своей части
   - сумму квадратов своей части  
4) `MPI_Reduce` собирает глобальные суммы на `rank 0`  
5) `rank 0` вычисляет:
   - mean = sum / N
   - std = sqrt( (sumsq/N) - mean^2 )

---

## Task 2 — Gauss (MPI_Scatter + MPI_Bcast + MPI_Gather)
1) `rank 0` создаёт матрицу `A (NxN)` и вектор `b (N)`  
2) строки делятся между процессами через `MPI_Scatter`  
3) прямой ход:
   - на каждом шаге k владелец строки k формирует pivot-строку
   - `MPI_Bcast` рассылает pivot всем процессам
   - каждый процесс зануляет элементы ниже диагонали в своих строках  
4) сбор результата на `rank 0` через `MPI_Gather`  
5) `rank 0` делает обратный ход и выводит решение

---

## Task 3 — Floyd-Warshall (MPI_Scatter + MPI_Allgather + MPI_Gather)
1) `rank 0` создаёт матрицу смежности `G (NxN)`  
2) строки делятся через `MPI_Scatter`  
3) на каждой итерации k:
   - каждый процесс обновляет свои строки
   - затем делается `MPI_Allgather`, чтобы у всех снова была общая матрица  
4) итог собирается на `rank 0` через `MPI_Gather`  
5) `rank 0` печатает кусок матрицы (например 8x8)
