# Task 1 — Global Memory vs Shared Memory (CUDA)

## Цель
Сравнить производительность поэлементной обработки массива на GPU в двух вариантах:
1) только с использованием глобальной памяти;
2) с использованием разделяемой (shared) памяти.

## Что сделано
- Создала массив `a` размером `N = 1 000 000`.
- Выполнила операцию: `a[i] = a[i] * k`, где `k = 2`.
- Замеряла время выполнения каждого CUDA-ядра через `cudaEvent`.

## Ожидаемый результат
- Вывод времени выполнения для двух вариантов.
- Проверка корректности: после обработки все элементы массива увеличены в `k` раз.
