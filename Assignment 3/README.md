# Assignment 3. Архитектура GPU и оптимизация CUDA-программ

Данный ассаймент посвящён изучению архитектуры GPU и основным приёмам оптимизации CUDA-программ.  
В работе рассматривается влияние различных типов памяти, шаблонов доступа к данным и параметров конфигурации сетки и блоков потоков на производительность программ, выполняемых на GPU.

Все задания реализованы и протестированы в среде Google Colab с использованием CUDA.

В файле `Control questions for Assignment 3` приведены ответы на контрольные вопросы

Также приложен файл `*.ipynb` со всеми заданиями и их выводами.

## Структура проекта

Проект логически разделён на 4 папки, каждая из которых соответствует отдельному заданию ассаймента.

## Содержимое каждой папки (task1 – task4)

В каждой папке задания содержатся следующие материалы:

- `*.cu` — исходный код CUDA-программы  
- `README.md` — краткое описание задания, идеи решения и полученных результатов  
- `task _ block scheme.md` — блок-схема алгоритма  
- `output task _.png` — скриншот вывода программы
