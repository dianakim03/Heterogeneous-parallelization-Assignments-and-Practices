# Task 4  

### Цель задания
Реализовать распределённую программу с использованием MPI для обработки массива:
- разделить массив между процессами,
- выполнить вычисления локально на каждом процессе,
- собрать результаты в один итог,
- замерить время выполнения для 2, 4 и 8 процессов.

В данной работе выполняется вычисление суммы элементов массива.


## Используемая среда
- Google Colab
- MPI: OpenMPI
- Язык: C++
- Компилятор: `mpic++`
- Запуск: `mpirun`

Важно: в Google Colab запуск MPI выполняется с параметрами:
- `--allow-run-as-root` (так как Colab работает от root)
- `--oversubscribe` (так как CPU-ресурсы ограничены, но нам нужно 2/4/8 процессов)

## Описание решения

### 1) Инициализация MPI
- `MPI_Init` запускает MPI среду.
- `MPI_Comm_rank` получает номер процесса (`rank`).
- `MPI_Comm_size` получает количество процессов (`size`).

### 2) Подготовка данных
- Задаётся размер массива `N = 1 000 000`.
- Каждый процесс будет получать `local_n = N / size` элементов.
- Общий массив `data` создаётся только на процессе rank = 0 и заполняется единицами.

### 3) Распределение массива между процессами (MPI_Scatter)
Используется `MPI_Scatter`:
- процесс `rank = 0` отправляет куски массива,
- каждый процесс получает свой кусок в `local_data`.

### 4) Локальные вычисления
Каждый процесс:
- проходит по своему массиву `local_data`,
- вычисляет `local_sum` (локальная сумма элементов).

### 5) Сбор результата (MPI_Reduce)
Используется `MPI_Reduce`:
- все `local_sum` суммируются,
- итог записывается в `global_sum` на процессе `rank = 0`.

### 6) Замер времени
- Перед вычислениями выполняется `MPI_Barrier` для синхронизации.
- Замер начинается перед `MPI_Scatter`.
- Замер заканчивается после `MPI_Reduce`.
- Время выводится только процессом `rank = 0`.
