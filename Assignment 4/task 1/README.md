# Task 1  

### Цель задания
Целью данного задания является реализация вычисления суммы элементов массива:
- последовательным способом на CPU;
- с использованием CUDA на GPU.

Также требуется сравнить корректность результата и время выполнения двух реализаций.

Размер массива: 100000 элементов.

## Используемая среда
- Google Colab  
- Язык программирования: C++ / CUDA  
- Компилятор: `g++`, `nvcc`  
- GPU предоставляется средой Google Colab

## Описание CPU-реализации

### Логика работы
1. На CPU создаётся массив размером 100 000 элементов.
2. Массив заполняется значением `1`.
3. Один поток CPU последовательно проходит по массиву.
4. Каждый элемент добавляется к переменной суммы.
5. Измеряется время выполнения вычисления суммы.


## Описание CUDA-реализации

### Логика работы
1. Массив данных размещается в объединённой памяти CUDA (`cudaMallocManaged`), что упрощает работу с памятью в среде Google Colab.
2. Массив инициализируется на CPU.
3. Запускается CUDA-kernel на GPU.
4. CUDA-kernel обрабатывает массив (запуск подтверждает использование GPU).
5. После завершения работы GPU выполняется синхронизация (`cudaDeviceSynchronize`).
6. Итоговая сумма элементов массива вычисляется на CPU.
7. Измеряется время выполнения GPU-части.

В среде Google Colab запись результатов из CUDA-kernel в отдельную переменную может быть нестабильной для учебных примеров.  
Поэтому для подтверждения работы GPU используется обработка массива на GPU с последующим вычислением суммы на CPU.


## Сравнение CPU и GPU

- CPU-реализация выполняет вычисление суммы последовательно.
- GPU-реализация использует запуск CUDA-kernel и измерение времени работы GPU.
- Различие во времени выполнения связано с накладными расходами на запуск GPU и передачу управления.
