# Task 1 — Vector Addition (CPU vs OpenCL GPU)

## Описание
В данной задаче реализовано сложение двух одномерных массивов.
Вычисления выполняются двумя способами: на CPU и на GPU с использованием OpenCL.
Основная цель — сравнить время выполнения и корректность результатов.

## Что было сделано
- Создаются два массива A и B и заполняются случайными значениями.
- Реализуется последовательное сложение массивов на CPU с использованием обычного цикла.
- Реализуется параллельное сложение массивов на GPU с использованием OpenCL kernel.
- Для GPU создается контекст, очередь команд и буферы памяти.
- Данные копируются из памяти CPU в память GPU.
- Каждый поток GPU складывает один элемент массивов.
- Результат копируется обратно в память CPU.
- Выполняется проверка корректности результатов.
- Измеряется и выводится время выполнения CPU и GPU версий.

## Принцип работы
Каждый элемент массива обрабатывается независимо.
Это позволяет эффективно распараллелить вычисления на GPU, где каждый поток выполняет одну операцию сложения.

## Результат
Программа выводит таблицу, в которой указаны:
- размер массива;
- время выполнения на CPU;
- время выполнения на GPU;
- корректность результатов для CPU и GPU.
