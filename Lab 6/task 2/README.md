# Task 2 — Matrix Multiplication (CPU vs OpenCL GPU)

## Описание
В данной задаче реализовано умножение двух матриц.
Вычисления выполняются на CPU и на GPU с использованием OpenCL.
Цель задачи — показать разницу во времени выполнения при последовательных и параллельных вычислениях.

## Что было сделано
- Задаются размеры матриц N, M и K.
- Создаются и заполняются матрицы A и B случайными значениями.
- На CPU умножение матриц выполняется с помощью трех вложенных циклов.
- На GPU используется OpenCL kernel, где каждый поток вычисляет один элемент результирующей матрицы.
- Создается OpenCL контекст, очередь команд и буферы памяти.
- Матрицы копируются из памяти CPU в память GPU.
- Результат вычислений копируется обратно на CPU.
- Выполняется сравнение результатов CPU и GPU.
- Измеряется и выводится время выполнения.

## Принцип работы
Каждый элемент результирующей матрицы вычисляется независимо от других.
Это позволяет распараллелить вычисления на GPU, где каждый поток отвечает за один элемент матрицы.

## Результат
Программа выводит таблицу с:
- размерами матриц;
- временем выполнения на CPU;
- временем выполнения на GPU;
- результатом проверки корректности вычислений.
