### **Задача 2. Работа с массивами и OpenMP**
Практическое задание
Файл: `Task2.ipynb`  
Программа:
- создаёт массив из 10000 случайных чисел;
- находит `min` и `max` последовательно и параллельно (OpenMP);
- сравнивает время выполнения и выводит выводы.

### **Задача 3. Параллельная сортировка с OpenMP**
Практическое задание
Файл: `Task3.ipynb`  
Программа:
- реализует сортировку выбором последовательно;
- добавляет параллелизм с OpenMP;
- проверяет производительность для размеров 1000 и 10000.

### **Задача 4. Сортировка на GPU с использованием CUDA**
Практическое задание
Файл: `Task4.ipynb`  
Программа:
- делит массив на подмассивы (каждый обрабатывается отдельным CUDA-блоком);
- выполняет поэтапное слияние отсортированных подмассивов;
- замеряет производительность для размеров 10000 и 100000.
