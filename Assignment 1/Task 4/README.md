# Task 4 — Вычисление среднего значения массива (последовательно и OpenMP reduction)

## Задание 4 (25 баллов)
Создайте массив из 5 000 000 чисел и реализуйте вычисление среднего значения
элементов массива последовательным способом и с использованием OpenMP с редукцией.
Сравните время выполнения обеих реализаций.

## Описание
Данная программа демонстрирует вычисление среднего значения элементов большого массива.
В программе создаётся массив из 5 000 000 чисел, который заполняется случайными значениями.
Затем среднее значение вычисляется двумя способами: последовательно и параллельно с
использованием OpenMP и механизма reduction. Также измеряется и сравнивается время
выполнения обеих реализаций.

## Этапы работы программы
1. Создание массива из 5 000 000 целых чисел
2. Заполнение массива случайными значениями
3. Последовательное вычисление суммы и среднего значения
4. Параллельное вычисление суммы и среднего значения с использованием OpenMP reduction
5. Измерение и сравнение времени выполнения последовательной и параллельной реализаций
