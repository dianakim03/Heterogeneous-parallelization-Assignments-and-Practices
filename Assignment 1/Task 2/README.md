# Task 2 — Поиск минимального и максимального элементов массива

## Задание 2 (25 баллов)
Создайте массив из 1 000 000 целых чисел и реализуйте последовательный алгоритм
поиска минимального и максимального элементов. Замерьте время выполнения алгоритма.

## Описание
Данная программа демонстрирует последовательную обработку массива большого размера.
В программе создаётся массив из 1 000 000 целых чисел, который заполняется случайными
значениями. После этого последовательно выполняется поиск минимального и максимального
элементов массива, а также измеряется время выполнения алгоритма.

## Этапы работы программы
1. Создание массива из 1 000 000 целых чисел
2. Заполнение массива случайными значениями
3. Последовательный поиск минимального элемента массива
4. Последовательный поиск максимального элемента массива
5. Измерение времени выполнения алгоритма
