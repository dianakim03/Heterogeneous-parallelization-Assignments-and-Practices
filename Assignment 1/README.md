# Assignment 1 — Динамическая память и параллельная обработка массивов

## Описание
Данный ассаймент посвящён изучению работы с динамической памятью в языке C++,
последовательной и параллельной обработке массивов, а также использованию
технологии OpenMP. В рамках ассаймента реализованы четыре программы,
демонстрирующие различные способы обработки массивов и измерения времени
выполнения алгоритмов.

В ходе выполнения заданий используются динамические массивы, последовательные
алгоритмы и параллельные алгоритмы с применением OpenMP.

## Задания ассаймента
Задание 1. Динамическое выделение массива и вычисление среднего значения элементов
Задание 2. Последовательный поиск минимального и максимального элементов массива
Задание 3. Параллельный поиск минимального и максимального элементов с использованием OpenMP
Задание 4. Вычисление среднего значения массива последовательным способом и с использованием OpenMP с редукцией

## Структура ассаймента
Содержится файл `Control questions for Assignment 1` с контрольными вопросами и ответами к ним
Каждый таск содержит:
1. исходный код программы с комментариями каждой строчки
2. скриншот вывода программы в консоль
3. `README.md` с описанием задания 

## Вывод
В результате выполнения Assignment 1 были изучены основы работы с динамической
памятью, последовательной и параллельной обработкой массивов, а также принципы
использования технологии OpenMP для ускорения вычислений.
