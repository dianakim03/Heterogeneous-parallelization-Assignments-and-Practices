# Task 1 — Динамический массив и вычисление среднего значения

## Задание 1 (25 баллов)
Реализуйте программу на C++, которая динамически выделяет массив из 50 000 целых
чисел, заполняет его случайными значениями от 1 до 100, вычисляет среднее значение
элементов массива и корректно освобождает выделенную память


## Описание
Данная программа демонстрирует работу с динамической памятью в языке C++.  
В программе динамически выделяется массив из 50000 целых чисел, который
заполняется случайными значениями в диапазоне от 1 до 100.  
После этого вычисляется среднее значение элементов массива и корректно
освобождается выделенная память.

## Этапы работы программы
1. Динамическое выделение массива из 50000 целых чисел
2. Заполнение массива случайными числами от 1 до 100
3. Вычисление среднего значения элементов массива
4. Освобождение динамически выделенной памяти
