# Task 3 — Параллельный поиск минимального и максимального элементов (OpenMP)

## Задание 3 (25 баллов)
Используя OpenMP, реализуйте параллельный поиск минимального и максимального
элементов массива из задания 2. Сравните время выполнения последовательной и
параллельной реализаций.

## Описание
Данная программа демонстрирует использование технологии OpenMP для параллельной
обработки массива. В программе выполняется поиск минимального и максимального
элементов массива размером 1 000 000 целых чисел как последовательным, так и
параллельным способом. Также производится сравнение времени выполнения двух
реализаций.

## Этапы работы программы
1. Создание массива из 1 000 000 целых чисел
2. Заполнение массива случайными значениями
3. Последовательный поиск минимального и максимального элементов массива
4. Параллельный поиск минимального и максимального элементов с использованием OpenMP
5. Измерение и сравнение времени выполнения последовательной и параллельной реализаций
